# **App Name**: Health Journey

## Core Features:

- Interactive Dashboard: User-friendly dashboard to visualize daily stats, goal progress, and mood/symptom logs.
- Customizable Logging: Log health data to capture the number of steps and water intake, as well as a way to create user customizable health measurements.
- Goal Setting: Ability to set personalized goals (e.g., steps per day, water intake, target mood) and monitor progress through visual aids.
- Mood and Symptom Tracking: Add a logging method for capturing their daily moods/symptoms by having customizable entries.
- Medical History Input: Section for inputting allergies, medical conditions, past surgeries, and current medications, to improve insight tool.

## Style Guidelines:

- Primary color: Fresh green (#77DD77) to evoke health and vitality.
- Background color: Light gray (#F0F0F0), for a clean and modern backdrop.
- Accent color: A slightly brighter, analogous green (#89e189) will provide a nice contrast for the CTAs.
- Body and headline font: 'Inter' sans-serif, for a modern and easily-readable design.
- Simple, minimalist icons to represent different tracking metrics and goals.
- Clean and intuitive layout, prioritizing ease of use and clear data visualization.
- Subtle animations to provide feedback and enhance the user experience.
